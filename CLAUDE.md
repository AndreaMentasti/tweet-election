## Machine Setup (One-Time per Collaborator)

1. **Python:** `cp code/py/config_local.py.template code/py/config_local.py`
   Edit `config_local.py` в†’ set `DATA_ROOT` to your Dropbox `data/` path.
2. **Stata:** Edit `global root` in `0-data_prep.do` в†’ your local repo path.
   `cp code/stata/config_local.do.template code/stata/config_local.do`
   Edit `config_local.do` в†’ set `global data_root` to your Dropbox `data/` path.
3. **Verify:** `python code/py/01_data_prep.py` should complete without errors.

Neither `config_local.py` nor `config_local.do` should ever appear in `git status`.

## Python Patterns

| Pattern | Idiom | Notes |
|---------|-------|-------|
| Paths | `import config as cfg` then `cfg.DATA_PROCESSED` | Never hardcode paths |
| Save figure | `fig.savefig(cfg.FIGURES / "name.png", dpi=300, bbox_inches='tight')` | Always 300 DPI |
| Save table | `df.to_latex(cfg.TABLES / "name.tex")` | Output to `output/tables/` |
| Seed | `random.seed(42)` at top of script | Only if stochastic |

## Stata Patterns

| Pattern | Idiom | Notes |
|---------|-------|-------|
| Root path | `global root "path/to/project"` at top | All paths via `$root` |
| LaTeX table | `esttab ... using "$root/output/tables/table.tex", booktabs label replace` | |
| PNG figure | `graph export "$root/output/figures/fig.png", replace width(2400)` | в‰€ 300 DPI @ 8in |
| Seed | `set seed 42` | Before any random draws |
